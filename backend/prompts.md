# LLM 提示词配置

本文档包含所有用于 LLM API 调用的提示词模板。

---

## 1. 单词生成提示词

用于根据输入单词生成视觉隔离的四选项及对应图像提示词。

### 模板

```
根据输入英文单词"{word}"，生成视觉隔离的四选项及图像提示词，JSON格式输出。

## 选项规则（重要：所有单词必须是英文）
| 选项 | 规则 |
|------|------|
| A | 原词"{word}"，展现核心含义 |
| B | 英文形近词（增/删/改≤3字母），用最具体字面含义，禁止直接通过词性转换修改单词 |
| C | 必须从单词 A 和 B 相关联的类别 中选择 1 个英文具象名词 |
| D | 必须从单词 A 和 B 相关联的类别 1 个英文具象名词 |
**重要提醒：words字段中的所有单词必须是英文单词，不能是中文！** |

## 强制随机性约束
- C 和 D 必须来自不同类别，且不能重复
- C、D 不得与 A、B 形成明显联想、上下位、同场景、同功能关系
- C、D 单词简单易懂，避免生僻词、罕见词、专业术语和抽象词，普通英语学习者能够直接理解。
- C、D 不得与最近 N 次历史输出中的任意单词重复（如果没有历史，可忽略）
- 若模型生成的 C 或 D 与历史重复、过于常见、过于抽象，必须重新生成，直到满足要求
- words 字段中的所有单词必须是单个英文单词，不能是中文、也不能是单词词组

## 视觉隔离（强制）
- 物体互斥：四图无重复物体
- 场景互斥：背景环境区分（室内/室外/抽象/纯色）
- 色彩互斥：主色调差异明显
- 逻辑互斥：无因果/对立/联想关系

## 提示词规范
- 统一艺术风格：Norman Rockwell插画风格，温暖怀旧的美式插画风格，细腻的人物表情和场景细节，特写镜头
- 中文描述：主体+背景+光影色调
- 聚焦字面含义，避免抽象
- 末尾添加`--排除 [其他选项元素]`

## 输出格式
```json
{
  "words": {
    "A": "{word}",
    "B": "与A英文形近词、但不重复、与其无明显联想的英文具象名词",
    "C": "与A和B相关联类别、但不重复、与其无明显联想的英文具象名词1",
    "D": "与A、B相关联类别、但不重复、与其无明显联想的英文具象名词2"
  },
  "meanings": {
    "A": "中文词义",
    "B": "中文词义",
    "C": "中文词义",
    "D": "中文词义"
  },
  "prompts": {
    "A": "中文提示词+--排除B/C/D元素",
    "B": "中文提示词+--排除A/C/D元素",
    "C": "中文提示词+--排除A/B/D元素",
    "D": "中文提示词+--排除A/B/C元素"
  }
}
```

仅输出JSON，无其他文字。words必须是英文单词，meanings必须是单词对应的中文直译，且尽量精简。图像提示词要求能够反映英文单词对应的视觉意象，并且对应视觉意象占据图像画面的主要部分。使用特写镜头突出单词对应视觉主体。
```

## 2. 单图提示词生成

用于为单个单词生成图像提示词。

### 模板

```
为单词"{word}"（选项{word_type}）生成图像提示词。

要求：
- 中文描述：主体物体+背景环境+光影色调
- 聚焦字面含义的具体视觉呈现，并且对应视觉意象占据图像画面的主要部分。
- Norman Rockwell插画风格，温暖怀旧的美式插画风格，细腻的人物表情和场景细节
- 避免抽象概念
- 使用特写镜头突出单词对应视觉主体
- 

仅输出提示词，无其他文字。
```

---

## 3. 视觉隔离检查清单

生成后需验证以下条件：

- [ ] **物体互斥**：四图中不出现重复元素
- [ ] **场景互斥**：背景环境无重叠
- [ ] **逻辑互斥**：无因果/对立/联想关系
- [ ] **B选项合规**：形变≤3字母，仅用字面含义
- [ ] **C选项合规**: 与单词A低关联、语义完全不同
- [ ] **D选项合规**: 与单词A、C低关联、语义完全不同
---

```json
{
  "words": {
    "A": "apple",
    "B": "apply",
    "C": "gun",

  "meanings": {

  }
}
```

## 5. 图像生成风格指南

### 推荐风格
- **Norman Rockwell插画风格**：温暖怀旧的美式插画风格，细腻的人物表情和场景细节

### 提示词结构
```
[主体物体] + [背景环境] + [光影效果] + [色调] + [艺术风格] + --排除 [其他元素]
```

### 示例
```
一个金色的钟表放在深色木桌上，柔和的顶光，暖黄调，Norman Rockwell插画风格 --排除 书本 咖啡杯 手机
```

## 6. 完形填空题目生成提示词

用于根据两个英文单词生成带干扰项的完形填空题目，包含中英文句子。

### 模板

```
你是一个英语教学专家。请根据以下两个英文单词生成一道完形填空题。

单词1: {word1}
单词2: {word2}

要求：
1. 生成一句中文句子作为题干，句子需要包含这两个英文单词的含义，在需要填入英文单词的位置用 ___ 表示留空（两个空格）
2. 生成一句英文句子作为题干，句子需要包含这两个英文单词，在需要填入英文单词的位置用 ___ 表示留空（两个空格）
3. 生成两个干扰选项（与句子语境相关但不是正确答案的英语单词），干扰选项应该是与主要单词词性相同、但意思不同的词
4. 提供两个单词的中文含义
5. 生成一个适合配图的图像提示词（中文），必须包含：Norman Rockwell插画风格，温暖怀旧的美式插画风格，细腻的人物表情和场景细节，特写镜头

请以JSON格式返回结果：
```json
{{
    "sentence": "中文句子，其中包含两个 ___ 作为留空位置",
    "sentence_with_answers": "完整的中文句子，包含填入的英文单词{word1}和{word2}",
    "sentence_en": "English sentence with two ___ blanks",
    "sentence_with_answers_en": "Complete English sentence with {word1} and {word2}",
    "word1_meaning": "单词1的中文含义",
    "word2_meaning": "单词2的中文含义",
    "distractor1": "干扰词1",
    "distractor2": "干扰词2",
    "distractor1_meaning": "干扰词1的中文释义",
    "distractor2_meaning": "干扰词2的中文释义",
    "image_prompt": "描述场景的图像生成提示词（中文），Norman Rockwell插画风格，特写镜头"
}}
```

注意：
- 句子应该自然流畅，符合日常使用场景
- 留空位置应该合理，使得填入对应单词后句子通顺
- 干扰词应该有一定的迷惑性，但不能是正确答案
- 英文句子和中文句子的意思应该对应
- 图像提示词应该描述句子所表达的场景
- 单词含义应该简洁明了
```

## 8. 听力练习生成提示词

用于根据场景描述生成听力练习文本。

### 模板

```
你是一个英语教学专家。请根据以下场景描述生成一段适合听力练习的英文文本。

场景描述: {scene_description}

要求：
1. 生成一段英文文本，长度严格控制在30个单词以内
2. 文本内容要与场景描述相关
3. 语言自然流畅，适合英语学习者练习听力
4. 生成一个适合配图的图像提示词（中文），必须包含：Norman Rockwell插画风格，特写镜头

请以JSON格式返回结果：
```json
{{
    "text": "生成的英文文本",
    "image_prompt": "描述场景的图像生成提示词（中文）"
}}
```

注意：
- 英文文本应该简洁明了
- 避免使用过于复杂的词汇
- 图像提示词应该描述场景的主要元素
```
